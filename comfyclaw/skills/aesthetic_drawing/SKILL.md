# Skill: Aesthetic Drawing

## Description

This skill is designed to rewrite user prompts to align with the **expert-level aesthetic standards**. It transforms simple descriptions into multi-dimensional, professional-grade artistic instructions that maximize scores across all fine-grained aesthetic attributes.

- **Trigger when**: The user seeks an "aesthetic drawing," "high quality," "masterpiece," "professional art," "award-winning," or when the prompt requires a sophisticated artistic depth that a simple description cannot provide.
- **Do NOT trigger when**: The user requests a minimalist icon, a simple technical diagram, or a low-fidelity "lo-fi" aesthetic where professional complexity is undesirable.

---

## Instructions

When this skill is active, rewrite the user's prompt by integrating the following eight aesthetic dimensions derived from the ArtiMuse expert evaluation system:

1. **Composition & Design (Spatial Balance)**

- Establish a clear visual hierarchy and dynamic focal points.
- Use professional layout techniques like leading lines, rule of thirds, or asymmetric balance to guide the viewer’s eye.

2. **Visual Elements & Structure (Color & Light)**

- Specify the interplay of contrasting or harmonious color palettes.
- Describe precise illumination (e.g., volumetric lighting, cinematic shadows) to enhance structural clarity and dimensionality.

3. **Technical Execution (Craftsmanship)**

- Define the mastery of the medium, whether it’s "precise brushwork and ink application" for paintings or "sharpness, focus, and subsurface scattering" for digital 3D renders.
- Emphasize high-resolution details and refined textures.

4. **Originality & Creativity (Conceptual Breakthrough)**

- Introduce surreal elements or imaginative crossovers that exceed common styles.
- Challenge conventional representations with fresh perspectives or experimental concepts.

5. **Theme & Communication (Narrative Clarity)**

- Infuse the prompt with a clear narrative or symbolic meaning.
- Ensure the subject effectively communicates a specific theme, cultural significance, or societal context.

6. **Emotion & Viewer Response (Atmospheric Impact)**

- Use sensory adjectives to evoke strong emotional responses (e.g., "tranquil mystery," "vibrant energy," "melancholic nostalgia").
- Create a lasting impression through a carefully curated atmosphere.

7. **Overall Gestalt (Harmonious Unity)**

- Ensure all elements (style, color, subject) blend into a cohesive and unified "Gestalt."
- The final image should feel like a singular, well-integrated work of art rather than a collection of random parts.

8. **Artistic Depth (Comprehensive Evaluation)**

- Add layers of complexity and nuanced details that reward close scrutiny.
- Aim for a "masterful" finish that demonstrates both artistic impact and intellectual depth.

## Output Format

Return ONLY the final enhanced prompt text. Do not include any conversational filler, introductory remarks, or prefixes like "Enhanced prompt:".
